Instrucciones para usar el logo adjunto

1) Guardar la imagen que has adjuntado en la conversación como `logo.png` dentro de la carpeta `public/`.

2) (Opcional pero recomendado) Generar versiones redimensionadas para mejor soporte:
   - `public/logo-192.png` (192x192)
   - `public/logo-512.png` (512x512)

   Puedes usar herramientas online (ej. https://squoosh.app/ o https://resizeimage.net/) o `preview` en macOS.

3) (Opcional) Generar `favicon.ico` si lo deseas y colócalo en `public/favicon.ico`.

4) Reinicia el servidor de desarrollo si está en ejecución (`npm run dev` o `pnpm dev`) para que Vite recargue los assets.

5) Verifica en el navegador abriendo la app y revisando la pestaña; limpia caché si no ves el cambio.
